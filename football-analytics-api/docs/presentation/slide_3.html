High Performance with Minimal Infrastructure: FastAPI + SQLAlchemy + SQLite

FastAPI
  • Chosen for its async performance (benchmarks show 2-3x faster than Flask).
  • Native OpenAPI/Swagger documentation generation.
  • Pydantic-based type-safe validation ensures robust data handling.
SQLAlchemy + SQLite
  • SQLite provides zero-configuration setup with full SQL support.
  • SQLAlchemy ORM abstracts database operations, preventing SQL injection.
  • Enables clean, relational data modelling.
JWT + bcrypt
  • JWT tokens (python-jose) provide stateless, industry-standard security.
  • Passwords hashed via bcrypt (passlib).
  • Public GET endpoints open; POST/PUT/DELETE require valid tokens.

Comprehensive Testing

Utilizes pytest with httpx TestClient for isolated integration testing against an in-memory database. 45 automated tests cover all CRUD operations, authentication flows, analytics logic, and error handling.

Layered Architecture

Design separates Routers → Schemas → Models → Database, strictly following separation of concerns and enabling independent testing and maintenance of each layer.
